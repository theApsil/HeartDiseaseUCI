# HeartDiseaseUCI
Multiclass classification of the presence of heart disease based on https://www.kaggle.com/datasets/redwankarimsony/heart-disease-data/data

## Описание проекта
Проект представляет собой решение задачи мультиклассовой классификации наличия сердечно-сосудистых заболеваний на основе набора данных Heart Disease UCI.

## Содержание проекта

### 1. Описание датасета
* **Источник данных**: UCI Machine Learning Repository
* **Объем**: 920 записей о пациентах
* **Признаки**: 76 атрибутов, включая демографические данные, результаты обследований и показатели ЭКГ
* **Классы**: 5 категорий (0-4)

### 2. Используемые модели
* **XGBoost** - градиентный бустинг
* **Random Forest** - случайный лес
* **Decision Tree** - дерево решений
* **Logistic Regression** - логистическая регрессия

### 3. Результаты моделей
| Модель | Точность | F1 Macro | F1 Weighted |
|--------|----------|-----------|-------------|
| XGBoost | 0.860851 | 0.841288 | 0.850031 |
| Random Forest | 0.608696 | 0.338987 | 0.580307 |
| Decision Tree | 0.489130 | 0.287968 | 0.561464 |
| Logistic Regression | 0.586957 | 0.332450 | 0.561464 |

### 4. Особенности датасета
* **Целевая переменная**: num (5 классов)
* **Типы признаков**:
  * Числовые (возраст, давление, холестерин)
  * Категориальные (пол, тип боли)
  * Бинарные (сахар, стенокардия)
* **Проблемы данных**: пропуски, выбросы, несбалансированность классов
